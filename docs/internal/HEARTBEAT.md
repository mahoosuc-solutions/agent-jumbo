Checked items `[x]` are skipped. Uncheck them to re-run.

- [ ] Check system health and report any anomalies

## Communications

- [ ] Check for unread messages across all channels
- [ ] Summarize any pending notifications

## Maintenance

- [ ] Verify scheduled tasks are running on time
- [ ] Check for any failed workflow steps and report
