# /strategy:dashboard - Unified Goal-Centric Dashboard

View your complete life, business, and real estate strategy with cross-domain alignment scoring, conflict detection, and recommended actions.

## Quick Start

**View complete dashboard:**

```bash
/strategy:dashboard view
```

**Get executive summary:**

```bash
/strategy:dashboard summary
```

**See active conflicts:**

```bash
/strategy:dashboard conflicts
```

**Get recommended actions:**

```bash
/strategy:dashboard actions
```

---

## System Overview

This command integrates ALL three domains into a **unified goal-centric view**:

1. **Life Goals** (Foundation): Personal values, vision, OKRs
2. **Business Goals** (Alignment): Software business serving life goals
3. **Real Estate Goals** (Alignment): Property investments serving life goals

**Key Principle**: Everything flows from your life goals. Business and real estate are tools to achieve those goals, not ends in themselves.

## Data Integration

Dashboard pulls data from all other commands:

- `/life:goals` - Life goals and OKRs
- `/life:habits` - Daily progress toward goals
- `/life:time` - Time allocation and goal alignment
- `/life:wellness` - Health metrics correlating with productivity
- `/software-business:projects` - Project revenue and profitability
- `/software-business:clients` - Client pipeline and satisfaction
- `/software-business:revenue` - Financial performance
- `/software-business:team` - Team capacity and progress
- `/real-estate:portfolio` - Property values and equity
- `/real-estate:tenants` - Occupancy and rent collection
- `/real-estate:maintenance` - Property condition
- `/real-estate:deals` - New acquisition analysis
